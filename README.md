# The website for Sapporo.js

http://sapporojs.org

## Environments

* Mac OS X Lion
* Ruby 2.0.0

## Setup

```sh
$ git clone git://github.com/sapporojs/sapporojs.org.git
$ cd sapporojs.org
```

Using [Bundler](http://gembundler.com/).

```sh
$ bundle install
```

## Boot

```sh
$ bundle exec middleman server
```
